# HimShakti ListingAI

> An AI-powered product description generator for small food processing businesses — built for e-commerce platforms like Amazon and Flipkart.

---

## About the Project

HimShakti ListingAI helps small food processing entrepreneurs generate compelling, keyword-rich product descriptions instantly. The user inputs basic product details and receives AI-generated copy with options to adjust tone and regenerate output — no copywriting skills needed.

**Sector:** Food Processing  
**Target Platforms:** Amazon, Flipkart

---

## Tech Stack

| Layer      | Technology              |
|------------|-------------------------|
| Frontend   | React JS                |
| Backend    | Node.js + Express.js    |
| Database   | MongoDB via Atlas       |
| AI / LLM   | Groq API (LLaMA 3)      |

---

## Core Features

- **Product Input Form** — Enter product name, ingredients, weight, and feature list
- **Tone Selector** — Choose from Premium, Traditional, or Health-Focused tones
- **AI-Generated Descriptions** — Instant copy via Groq LLM API with custom system prompts
- **Regenerate Button** — Get alternate outputs with one click
- **Copy to Clipboard** — Directly paste into Amazon / Flipkart listing fields

---

## AI Integration

LLM API integration using **Groq** with a custom system prompt that generates SEO-optimized, tone-controlled product descriptions based on user input. Prompt engineering ensures output stays relevant, natural, and platform-ready.
